## If

```tsx
$<vivi>
  if(n == 1) {
      if(n == 2) {
          return 2;
      }
      return 1;
  }
$</vivi>
```

## If Else

```tsx
$<vivi>
  if(n == 1) {
    return 1;
  } else {
    return 2;
  }
$</vivi>
```

## For

```tsx

  for(n: int = 1; a < 10; n) {

  }
$</vivi>
```
